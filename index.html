Everything runs locally in your browser. Audio is never uploaded. Speaker labels aren't automatic — add them in the field beside each line if you want them.
How it works. Whisper runs on-device via Transformers.js (WebGPU when available, otherwise CPU). Nothing leaves this page. Drop a single file to trim and transcribe a span, or drop up to 20 at once (e.g. WhatsApp .ogg / .opus voice notes) to transcribe them all in one run — each appears as its own section with its own export. For one long recording, trim to the part you need; a 1-hour span on the base model takes a few minutes with WebGPU, longer on CPU.
